Below are the commands for your NPC. The first in each group runs in a repeat command block, the others in chain.

Commands for Onk the Explorer (node 1)

/execute as @a[scores={onk1=1}] run tellraw @s [{"text":"[Onk the Explorer]","color":"gold"},{"text":" Greetings my fellow adventurer! Is there a humble quest on which you wish to embark today?","color":"white"}]
/execute as @a[scores={onk1=1}] run tellraw @s {"text":"> [Show me what you got!]","color":"light_purple","click_event":{"action":"run_command","command":"/trigger onk2"}}
/execute as @a[scores={onk1=1}] run tellraw @s {"text":"> [No thank you!]","color":"light_purple","click_event":{"action":"run_command","command":"/trigger onk3"}}
/execute as @a[scores={onk1=1}] run scoreboard players set @s onk1 0
